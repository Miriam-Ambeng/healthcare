# Health Care Appointment Frontend

React frontend for the Automated Health Care Appointment System backend.

## Setup

```bash
npm install
npm run dev
```

Open the URL printed by Vite, usually:

```text
http://127.0.0.1:5173
```

Make sure the backend is running on:

```text
http://localhost:5000
```

## Test Flow

1. Register a doctor.
2. Login as the doctor and save a schedule.
3. Sign out.
4. Register or login as a patient.
5. Choose the doctor, pick a date that matches the doctor's schedule, select a slot, and book.
6. View appointments from the appointments tab.
